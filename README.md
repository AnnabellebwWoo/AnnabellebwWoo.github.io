# **Belle Reve - Personal Blog**

Custom built blog by Annabelle Woo, created with Next.js, Typescript, and CSS modules. Previous blog was hosted on WordPress, but rebuilding it myself allows me to have full design control. 

---
## **Tech Stack**

- **Next.js (App Router)**
- **TypeScript**
- **CSS Modules** (`*.module.css`) for styling
- **Vercel** for deployment
- **MDX** for writing posts
- **Prettier + ESLint**

---

## **Features**
- Fully responsive layout
- Custom components (Header, Footer, Blog Cards, Sidebar, etc.)
- Mbile hamburger menu with full-screen overlay
- MDX support for writing posts

---

## **Writing Posts**

Example MDX file:

```md
---
title: "Rebuilding My Blog From Scratch"
date: "2025-07-11"
category: "misc"
excerpt: "Behind the scenes of coding Belle Reve."
---

Your content here...
```
